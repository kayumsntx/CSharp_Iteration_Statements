1. while Loop:
Executes a block of code repeatedly as long as a condition is true. Condition is tested before each iteration.

2. for Loop:
Executes a block of code a specified number of times. Combines initialization, condition, and increment in one line.

3. do-while Loop:
Executes a block of code at least once, then repeats as long as condition is true. Condition is tested after each iteration.

4. foreach:
The foreach loop provides a simple, clean way to iterate through all elements in a collection (arrays, lists, dictionaries, etc.) without manually managing an index variable.
